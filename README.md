# Zodl Zcash Wallet

This repository is the home of the Zodl Zcash wallet. The Zodl wallet is
publicly available from the Apple Store and the Google Play Store.

At present, there are implementations of the Zodl wallet for Android and iOS;
platform-specific source code can be found here:
* Zodl for [Android](https://github.com/zodl-inc/zodl-android)
* Zodl for [iOS](https://github.com/zodl-inc/zodl-ios)

## Reporting an issue

If you'd like to report feature request for the Zodl Wallet, please file a
GitHub issue [here](https://github.com/zodl-inc/zodl-project/issues/new/choose).

If you'd like to report a bug you encountered in the Zodl Android Wallet, please
file a Github issue [here](https://github.com/zodl-inc/zodl-android/issues/new?labels=bug&template=bug-report.md).

If you'd like to report a bug you encountered in the Zodl iOS Wallet, please
file a GitHub issue [here](https://github.com/zodl-inc/zodl-ios/issues/new?labels=bug&template=bug-report.md).

If you wish to report a security issue, please follow our
[Responsible Disclosure guidelines](responsible_disclosure.md).
See the [Wallet App Threat Model](wallet_threat_model.md)
for more information about the security and privacy limitations of the wallets.
